HTML5 and CSS Normalize

HTML5 Popularity

HTML5 has quickly become extremely popular and will just keep skyrocketing, as it introduces many new and friendly functionalities, as well as making the creation of websites a whole lot easier. It has many new features such as better interactions (document editing, drag and drop, media playback), ability to write the code in a neat and readable manner, which is much easier to understand, and allowing the site to be displayed on multiples browsers with minimal differences. HTML5 allows programmers to easily create mobile versions of a site for on the go browsing, and also opens doors to creating online games. It can also support more ways to feature audio/video or other visuals on the site.


Using "normalize.css"

"normalize.css" is an extremely popular file used by many big names such as Twitter, GitHub, Guardian, and many others. This CSS file allows for better cross-browser development, as it cancels not all, but most of the varying CSS styles that differ on browsers. This creates a consistent look for the website on most browsers. It is known as the modern HTML5-ready alternative to the traditional CSS reset. (http://nicolasgallagher.com/about-normalize-css/)
